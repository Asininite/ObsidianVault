### Reversing with a different array
```cpp
class Solution{
    public:
        vector<int> reverseArray(vector<int>& nums){            
            vector<int> temp;
            for(auto it = nums.end() - 1; it >= nums.begin(); --it){
                temp.push_back(*it);
            }
            return temp;
        }
};
```
