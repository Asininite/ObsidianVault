```java
class Basics{
	public static void main(String[] args){
		System.out.println(x:"Hello!");
	}
}
```

- command-line inputs goes into `String[] args`
- `Basics.main` is the entry point
- `void` means it has to return nothing

```java
byte num = 100;
// -128 to 127
// 8 bits in memory

short num2 = 200;
// -32768 to 32767
// 16 bits in memory

int num3 = 300;
// -2^31 to 2^31 - 1
// 32 bits in memory

float num5 = 500.0;
// 32 bits in memory

long num4 = 400;
// 64 bits in memory

double num6 = 600.0;
// 128 bits in memory
```

bool something = true;
