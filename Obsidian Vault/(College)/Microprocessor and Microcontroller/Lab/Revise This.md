- ROM : 8000:0000 - 8000:FFFF H
- RAM : 0000:0000 - 0000:FFFF H
- user can enter program from 0400H to FFFFH 
- 8255-1 :
	  PORT A : 40H
	  PORT B : 42H
	  PORT C : 44H
	  CONTROL WORD : 46H

## Menu Commands
- A - assemble
- B - block move
- D - disassemble
- F - fill data 
- G - go and execute
- M - view memory / modify memory
- E - edit / view memory
- R - register view / edit
- S - single step
- T - view / edit breakpoint
