## OS Module
- provides functions to interact with OS

1. os.name : gives name of OS-dependent system module
2. os.getcwd() : returns current working directory
3. os.listdir() : print files and directories
4. os.chdir()
5. os.mkdir(PATH)
6. os.rmdir(PATH)
7. os.remove(PATH)
8. os.rename(old.new)
