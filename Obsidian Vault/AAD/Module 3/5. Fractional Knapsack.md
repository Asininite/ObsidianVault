- total weight **not more** than knapsack capacity

```
algorithm knapsack (m, n){
	for i = 1 to n do{
		x[i] = 0.0;
		u = m;
	}

	for i = 1 to n do{
		if w[i] > u then {
			break;
		} else {
			x[i] = 1.0;
			u = u - w[i];

			if i < n then {
				x[i] = u/w[i];
			}
		}
	}
}
```
